# 🌍 Smart Travel Planner

## 🚀 Project Overview
Smart Travel Planner is a web application that generates personalized travel itineraries based on user inputs like location, budget, number of days, and interests.

---

## 🎯 Problem Statement
Planning trips is time-consuming and confusing. Users often struggle to create a proper itinerary within their budget.

---

## 💡 Solution
Our app simplifies travel planning by generating:
- Day-wise itinerary
- Budget breakdown
- Suggested places to visit

---

## ⚙️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- AI: Simulated (offline logic for reliability)

---

## 🧩 Features
- User input for location, budget, days, interest
- Instant travel plan generation
- Clean and simple UI
- Works offline (no API dependency)

---
