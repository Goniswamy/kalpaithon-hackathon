# 🌍 Smart Travel Planner

## 🚀 Overview
Smart Travel Planner is an AI-powered web application that generates personalized travel itineraries based on user preferences like location, budget, duration, and interests.

---

## 🎯 Problem Statement
Planning trips manually is time-consuming and confusing. Users struggle to:
- Choose destinations
- Manage budgets
- Plan daily itineraries

---

## 💡 Solution
Our system automatically generates:
- 📅 Day-wise travel plans
- 📍 Top tourist attractions
- 💰 Budget breakdown
- 🎯 Personalized recommendations

---

### Frontend

### Backend

- API / Local Model (Ollama / Gemini / Mock API)

## ⚙️ Features
- Smart itinerary generation
- Budget-based recommendations
- User-friendly UI
- Fast response system
- Customizable travel preferences

---

## 🧩 How It Works

1. User enters:
   - Destination
   - Number of days
   - Budget
   - Interests

2. Backend processes request

3. AI / logic generates travel plan

4. Output displayed in UI
