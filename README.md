# 🌍 Smart Travel Planner

---

## 🎯 Project Title  
Smart Travel Planner

---

## ❗ Problem Statement  
Planning trips manually is time-consuming and confusing. Users often struggle to:
- Choose the right destination  
- Manage their budget effectively  
- Plan a proper day-wise itinerary  

---

## 📖 Description  
Travel planning involves multiple decisions like selecting places, budgeting, and organizing daily activities. Many users lack proper guidance and end up wasting time searching across different platforms. This leads to inefficient planning, missed attractions, and poor budget management.

---

## 💡 Proposed Solution  
Smart Travel Planner is an AI-powered web application that simplifies travel planning by generating personalized itineraries based on user input.

It provides:
- 📅 Day-wise travel plans  
- 📍 Top tourist attractions  
- 💰 Budget breakdown  
- 🎯 Personalized recommendations  

---

## 🛠️ Tech Stack

### Frontend
- HTML  
- CSS  
- JavaScript  

### Backend
- Flask (Python)  

### AI / Logic
- API / Local Model (Ollama / Gemini / Mock API)  

---

## ⚙️ Features
- Smart itinerary generation  
- Budget-based recommendations  
- User-friendly UI  
- Fast response system  
- Customizable travel preferences  

---

## 🧩 How It Works
1. User enters:
   - Destination  
   - Number of days  
   - Budget  
   - Interests  

2. Backend processes the request  

3. AI / logic generates travel plan  

4. Output is displayed in the UI
